# Afkbot
Grand RP AFK Bot, verwenden auf eigene Gefahr!

Download [hier](https://github.com/tr1rd/afkbot/releases)

## Erklärung
### Verwendung
1. Die .exe Datei [hier](https://github.com/tr1rd/afkbot/releases) herunterladen
2. Doppelklick auf die Datei
3. GTA starten bzw. GTA fokussieren
4. F8 drücken um den Bot zu aktivieren

### Wie funktioniert er?
Der Bot hält nach der aktivation jeweils abwechselnd den W button (virtuell) gedrückt und lässt ihn wieder für eine Minute los, so dass, falls der Character bewusstlos wird, W immernoch gedrückt wird und nicht abgebrochen werden kann
